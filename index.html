What's Wrong With This UI?

Problem 1: Hard to See Important Stuff

The background is too dark and everything looks the same. You can't tell what's important and what's not. The required fields have tiny asterisks that are easy to miss.

How to fix it: Use a lighter background and make important fields stand out more. Make required field markers bigger and more obvious.

Problem 2: Confusing Buttons

The toggle switches and dropdowns don't look clickable. You can't tell if they're on or off easily.

How to fix it: Use regular checkboxes instead of toggle switches. Make buttons look more like buttons with clear labels.

Problem 3: Text is Too Small

A lot of the helper text and descriptions are really small and hard to read, especially the green success message.

How to fix it: Make the font size bigger (at least 16px for regular text). Use better contrast colors.

Problem 4: Bad Spacing

Everything is cramped together and there's not enough space between sections. It's hard to see where one section ends and another begins.

How to fix it: Add more padding and margin. Put boxes around related items to group them together.

Problem 5: Messy Layout

The form fields aren't aligned well. The owner dropdown and repository name are at different heights which looks weird.

How to fix it: Line everything up vertically. Stack form fields on top of each other instead of side by side.
